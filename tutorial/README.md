# Riot4 Tutorial

## Usage

First of all, plz execute this commands.

```cmd
$ cd ../
$ yarn install
$ yarn tutorial
```

Perhaps the browser will open automatically and the application should start.

URL: [http://127.0.0.1:8080/](http://127.0.0.1:8080/)
